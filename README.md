## 🔗 Cara Install
<br>

- <code>git clone ↆↆ </code>
  
  ```bash
  git clone https://github.com/bagussatoto/image-get-palette.git
  ``` 
  
- <code>cd image get palette</code>
- <code>npm install</code>
- <code>npm run dev</code>
